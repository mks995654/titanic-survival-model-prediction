# titanic-survival-model-prediction
Titanic survival prediction using supervised learning models with performance comparison, feature importance analysis, and evaluation metrics.

Project Objective-

This project focuses on evaluating machine learning models for predicting passenger survival on the Titanic dataset.
As a QA/Test Engineer transitioning into AI/ML Engineer, the goal of this project is to apply model testing techniques, performance evaluation,
and comparison of algorithms.

Models Implemented and Trained-

1️ . Logistic Regression
2️ . Random Forest

Model Evaluation Strategy -

The following testing metrics were used:
Accuracy
Confusion Matrix
Precision
Recall
F1 Score
Feature Importance
